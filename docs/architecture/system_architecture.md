**# System Architecture**


Cycloidal drives will be used to move each articulation. 


## Joint Configuration (Axis 1–3)

Motor → Belt Drive → Cycloidal Reducer → Encoder → Joint

These joints experience the highest loads.  
The belt stage increases the overall reduction ratio without increasing the size of the cycloidal drive.


## Joint Configuration (Axis 4–6)

Motor → Cycloidal Reducer → Encoder → Joint
